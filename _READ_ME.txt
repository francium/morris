-- MAC OS or Linux --
- To compile and run on unix systems, do
$ ./make.sh     # if it doesn't work, try $ sh make.sh
$ ./run.sh      # '' ''   ''     ''   ''  $ sh run.sh

- To run the precompiled jar file, do
$ jar -jar out/artifacts/morris_jar/morris.jar


-- Windows --
- To run the precompiled jar on windows, double click it
